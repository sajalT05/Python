Interpretor--->

1. interpreted language, which can save you considerable time during 
    program development because no compilation and linking is necessary. 
2. interpreter can be used interactively, 
    a. which makes it easy to experiment with features of the language, 
    b. to write throw-away programs, 
    c. or to test functions during bottom-up program development. 
    d. It is also a handy desk calculator.
3. Programs written in Python are typically much 
    shorter than equivalent C, C++, or Java programs, for several reasons:
        a. the high-level data types allow you to express 
            complex operations in a single statement;
        b. statement grouping is done by indentation 
            instead of beginning and ending brackets;
        c. no variable or argument declarations are necessary.
